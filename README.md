## Clone

```bash
git clone git@github.com:exactly-protocol/protocol.git
```

## Install Locally

```bash
npm install --legacy-peer-deps
```

## Run Tests

```bash
npm test
```

## Run Coverage

```bash
npm run coverage
```

## Gas Reports

```bash
export REPORT_GAS=1 && npm test
```

## Deploy

```bash
npm run deploy:rinkeby
```
