```bash
mkdir -p ~/dev/exactly && cd ~/dev/exactly
git clone git@github.com:exactly-finance/protocol.git
```

## Install Locally

```bash
npm install
```

## Run Tests

```bash
npx hardhat test
```

npx hardhat coverage
