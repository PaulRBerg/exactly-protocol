## Clone

```bash
git clone git@github.com:exactly-protocol/protocol.git
```

## Install Locally

```bash
yarn
```

## Run Tests

```bash
yarn test
```

## Run Coverage

```bash
yarn coverage
```

## Gas Reports

```bash
REPORT_GAS=1 yarn test
```

## Deploy

```bash
yarn deploy:goerli
```
